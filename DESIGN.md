# Lumen 设计文档

## 项目概述

**Lumen** 是一个个人生活与财富管理应用，核心理念是"你的生活，你的微光，你的平衡"。应用旨在帮助用户记录重要的人生节点，追踪财富变化，并规划人生目标，实现生活与财富的和谐统一。

### 核心功能

1. **人生叙事** - 记录并展示重要的人生节点与里程碑
2. **财富记录** - 月度财富追踪与趋势分析
3. **人生目标** - 目标设定与进度管理
4. **资产清单** - 资产配置与可视化展示

---

## 设计理念与哲学

### 核心设计原则

#### 1. 简约而深刻
- **视觉语言**：采用大量留白、优雅的字体和精致的排版
- **信息密度**：在保持界面简洁的同时，提供丰富的信息层次
- **色彩哲学**：温暖的渐变色调传达希望与成长

#### 2. 情感化设计
- **叙事性**：将数据转化为人生故事，而非冰冷的数字
- **情感收益**：每个节点都标注"情感收益"，强调非货币价值
- **温度感**：通过温暖的色调和柔和的动画创造情感连接

#### 3. 数据即故事
- **时间维度**：所有数据都带有时间戳，构成个人历史
- **关联性**：财富记录、人生节点、目标之间相互关联
- **可视化**：通过图表和进度条让数据一目了然

## 数据模型

### 核心数据类型

#### 1. Milestone (人生节点)
```typescript
interface Milestone {
  id: string;                    // 唯一标识
  date: Date;                    // 发生日期
  title: string;                 // 标题
  description: string;           // 详细描述
  category: MilestoneCategory;   // 类别
  emotionalYield: string[];      // 情感收益
  capitalDeployed: number;       // 投入资本
  assetClass: AssetClass;        // 资产类别
  imageUrl?: string;             // 关联图片
  location?: string;             // 地理位置
  status?: MilestoneStatus;      // 当前状态
}
```

**类别分类**：
- `life-chapter` - 人生章节
- `vision-realized` - 愿景实现
- `strategic-asset` - 战略资产
- `experience` - 人生体验
- `foundation` - 基础建设

#### 2. WealthRecord (财富记录)
```typescript
interface WealthRecord {
  id: string;                    // 唯一标识
  date: Date;                    // 记录日期
  totalAssets: number;           // 总资产
  changeAmount: number;          // 变化金额
  changeReason: string;          // 变化原因
  breakdown: AssetBreakdown;     // 资产分解
}
```

**资产分解**：
- `liquidCapital` - 流动资金
- `equities` - 股票与指数基金
- `realEstate` - 房地产
- `other` - 其他资产

#### 3. LifeGoal (人生目标)
```typescript
interface LifeGoal {
  id: string;                    // 唯一标识
  title: string;                 // 目标标题
  description: string;           // 目标描述
  category: GoalCategory;        // 目标类别
  targetDate?: Date;             // 目标日期
  progress: number;              // 进度百分比
  estimatedCost: number;         // 估算成本
  milestones: string[];          // 关联的节点ID
  status: GoalStatus;            // 当前状态
}
```

**目标状态**：
- `dreaming` - 梦想中
- `planning` - 规划中
- `in-progress` - 进行中
- `achieved` - 已实现

## 组件库

### 核心组件

#### 1. Sidebar (侧边栏)
**功能**：
- 应用Logo展示
- 导航菜单
- 全局指标显示

**交互特性**：
- 当前页面高亮显示
- 激活状态的小圆点指示器
- 固定在左侧的布局

#### 2. Modal (模态对话框)
**功能**：
- 统一的弹窗容器
- 支持表单与内容展示

**交互特性**：
- 背景点击关闭
- ESC键关闭
- X按钮关闭
- 打开时禁止背景滚动

#### 3. Progress (进度条)
**功能**：
- 可视化进度展示
- 动态颜色编码
- 百分比显示

**特性**：
- 三种尺寸：sm, md, lg
- 四种颜色：warm, blue, gold, green
- 自动颜色分级：
  - ≥75%: 绿色
  - ≥50%: 金色
  - ≥25%: 蓝色
  - <25%: 灰色
- 动画闪光效果
- 0%, 50%, 100% 刻度标记

#### 4. DatePicker (日期选择器)
**功能**：
- 中文本地化
- 样式定制匹配设计系统

#### 5. ChapterCard (章节卡片)
**功能**：
- 人生节点的卡片展示
- 图片与文本的交替布局

**交互特性**：
- Hover显示编辑/删除按钮
- 根据图片位置调整按钮位置

## 交互设计

### 统一交互模式

#### 1. CRUD操作
- **创建**：主操作按钮 + Modal表单
- **读取**：卡片/列表展示
- **更新**：Hover显示编辑按钮 + Modal表单
- **删除**：Hover显示删除按钮 + 确认对话框

#### 2. 表单交互
- **即时验证**：表单字段即时验证
- **取消机制**：取消按钮 + 背景点击 + ESC键
- **保存反馈**：保存后自动关闭Modal

#### 3. 导航交互
- **当前页面高亮**：激活菜单项的文本颜色 + 小圆点
- **平滑滚动**：页面内锚点平滑滚动
- **返回导航**：浏览器返回按钮支持

### 动画与过渡

#### 页面过渡
- **淡入效果**：页面切换时淡入
- **时长**：300ms

#### 卡片交互
- **悬停提升**：阴影增强 + 轻微上移
- **时长**：200ms

#### 进度条动画
- **填充动画**：700ms 缓动
- **闪光效果**：2秒无限循环

#### Modal动画
- **淡入淡出**：背景与内容同时淡入淡出
- **时长**：200ms
