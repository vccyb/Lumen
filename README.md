# Lumen - 你的生活，你的微光，你的平衡

Lumen 是一个结合人生记录、财富管理和目标追踪的个人管理应用。

## 核心理念

- **生活记录**: 记录人生中的重要节点事件和转折点
- **财富管理**: 不同于普通记账软件，关注每月的财富变化趋势和原因
- **目标追寻**: 将人生目标、财富管理和经历记录有机结合

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **字体**: Inter (Google Fonts)

## 项目结构

## 核心功能

### 1. 人生叙事 (Life Narrative)
- 以时间线形式展示人生重要节点
- 每个节点包含日期、标题、描述、情感回报和资本配置
- 支持图片展示和位置信息

### 2. 财富记录 (Wealth Ledger)
- 记录总资产变化
- 分析资产配置（流动资金、股票、房产、私募投资等）
- 展示财富增长趋势

### 3. 人生目标 (Life Goals)
- 设置和追踪人生目标
- 分类：财务、体验、个人成长、关系、传承
- 关联相关的人生节点

### 4. 资产清单 (Asset Inventory)
- 详细列出所有资产
- 按类别分组
- 显示当前估值

### 5. 愿景规划 (Vision Planning)
- 规划未来愿景
- 设定实现时间线
- 估算所需资本

## 设计系统

### 颜色
- **背景**: `#F4F4F6` (基础), `#FFFFFF` (表面)
- **文本**: `#111111` (主要), `#666666` (次要), `#999999` (第三)
- **强调**: `#F0A07A` (金色), `#E28B6B → #D5546C` (温暖渐变)
- **边框**: `#EAEAEA` (微妙), `#CCCCCC` (聚焦)

### 间距
- **xs**: 4px
- **sm**: 12px
- **md**: 24px
- **lg**: 48px
- **xl**: 96px
- **xxl**: 160px

### 圆角
- **sm**: 8px
- **md**: 12px
- **lg**: 16px

### 阴影
- **subtle**: `0 4px 24px rgba(0, 0, 0, 0.04)`
- **elevated**: `0 8px 32px rgba(0, 0, 0, 0.08)`
- **glow**: `0 0 80px 15px rgba(245, 165, 120, 0.25)`
